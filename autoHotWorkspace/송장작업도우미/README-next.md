# 송장작업도우미 Next

기존 `송장작업도우미.ahk`는 그대로 유지하고, 차세대 크롤링 기능은 `송장작업도우미-next.ahk`에서 실험합니다.

## 이번 버전 목표

네이버 주문 내역에서 `의자` 같은 키워드로 검색한 뒤, 현재 Chrome 화면에 보이는 주문 목록 중 송장번호가 있는 항목을 한 번에 추출해 리스트로 보여줍니다.

## 사용 흐름

1. 네이버를 선택하고 검색어에 `의자` 입력
2. `선택 검색`으로 네이버 주문 내역 검색 페이지 열기
3. 네이버 주문 목록에서 송장번호가 보이는 상태 확인
4. `크롤링` 또는 `Win+F4`
5. 결과 리스트에서 `주소키 | 택배사 | 송장번호` 확인
6. `선택복사` 또는 `전체복사`로 Excel 붙여넣기용 텍스트 복사
7. 결과가 맞으면 `엑셀입력` 또는 `Win+F5`로 리스트를 순회하며 Excel에 입력

## 설계 방향

- 크롤링 기능은 Excel 입력 기능과 분리합니다.
- Next 화면은 수동 입력칸이 아니라 크롤링 결과 리스트를 중심으로 구성합니다.
- 목록 크롤링은 먼저 결과 확인과 복사를 담당합니다.
- `엑셀입력`은 리스트에 쌓인 row를 순회하면서 기존 Excel 안전 입력 로직을 호출합니다.
- 나중에 추출 결과가 충분히 안정되면 `검색 → 크롤링 → Excel 입력`을 한 번에 처리하는 버튼을 추가합니다.

## 현재 한계

네이버 주문 목록 화면을 `Ctrl+A`, `Ctrl+C`로 복사한 텍스트에서 추출합니다. 페이지에 주소가 표시되지 않으면 주소키는 `주소?`로 표시됩니다. 첫 실사용에서 복사되는 텍스트 샘플에 맞춰 주소/주문 블록 분리 규칙을 튜닝하는 단계입니다.
